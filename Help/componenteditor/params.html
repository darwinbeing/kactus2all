Parameters editor

Parameters editor can be used to add, remove and modify parameters of a component. Parameters can be used to configure or hold information related to the containing component.

The name column contains the name of the parameter and description column the textual description of the parameter. Value contains the value of the parameter.

Parameters editor contains a context menu that provides following options:

  • Add new parameter (Add row)
  • Remove parameter (Remove row)
  • Clear the selected cells
  • Copy the contents of the selected cells to clipboard
  • Paste the contents of the clipboard to the selected cells
  • Import a csv-file to the editor
  • Export the contents of the editor to a csv-file
