Address block editor

Address block editor is used to edit the details of an address block by defining registers to it. Each row in the editor specifies a single register within the containing address block.

Each register has a name and a textual description. Offset specifies the location of the register from the start of the containing address block expressed as number of addressing units. Size defines the number of bits the register contains. Dimension assigns an unbounded dimension to the register. Volatile value of true indicates that register value may change without a write operation to it, i.e. by an interrupt event.

Access column specifies the accessibility of the register, the possible values are:

  • Read-write
  • Read-only
  • Write-only
  • Read-writeOnce
  • writeOnce
