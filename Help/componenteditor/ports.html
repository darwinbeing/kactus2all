Type and Type definition

Select from the drop-down list, or type a new custom type (not recommended).

Left (higher) and right (lower) bound

For a scalar port leave these empty. For a vector port, the higher and lower bound tell the width of the port and alignment of the bits. Note that the LSB can be either on the left or right border depending on the type.

Width = max (left,right) – min (left,right) +1.

Example: for a VHDL std_logic_vector(29 downto 3) is given as  left=29 and right=3
